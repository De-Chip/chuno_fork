# PROJECT_CHUNO
멋쟁이사자처럼 프론트엔드 스쿨 9기, 팀 놀9있조 개발.

## 🎶프로젝트 소개
프로젝트 명인 추노(CHUNO)에는 '**추**천하는 **노**래'라는 의미를 담아, 본 프로젝트는 사용자들로 하여금 각자의 플레이스트를 공유하는 경험을 유도하고 있습니다.
